# MLScientistNoLeetcodeTechQuestions

Collection of non-LeetCode technical practice questions for ML Scientist / ML Engineer interviews.

## Repository Contents

### Root
- `LICENSE`: Project license.
- `README.md`: Project overview and navigation.
- `src/`: Practice problems and notebooks.

### src/autograde
- `Autograde.ipynb`: Custom autograd exercises in PyTorch (ReLU and SiLU), gradient derivations, implementation, checks, and visualization.

### src/stablesoftmax
- `StableLogSoftMax.ipynb`: Notebook exploration of numerically stable softmax / log-softmax.
- `play_softmax.py`: Script for experimenting with softmax behavior and stability.

## Goal

Practice interview-style ML coding and math questions focused on:
- numerical stability,
- automatic differentiation,
- implementation details beyond standard coding puzzles.
